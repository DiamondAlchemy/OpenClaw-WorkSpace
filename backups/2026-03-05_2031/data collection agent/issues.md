# Issues Log

Open issues across all facilities.

## Priority Levels
- **Low:** Minor note, self-corrected deviation
- **Medium:** Unresolved issue, unusual yield, equipment concern
- **High:** Run stopped, equipment down, yield significantly off target
- **Critical:** Safety issue, compliance concern, total equipment failure
